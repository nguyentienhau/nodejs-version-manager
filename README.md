# NodeJS Version Manager

## Basic Commands

    nvm on
    nvm off
    nvm version
    nvm help
    nvm commands
    nvm install <version>
    nvm uninstall <version>
    nvm list [available]
    nvm use <version>
    nvm current

## Add Environment Variables

    NVM_HOME		C:\Users\Admin\Programs\NodeJS Version Manager
    NVM_CURRENT		18.20.4

## Add Paths

    %NVM_HOME%
    %NVM_HOME%\versions\%NVM_CURRENT%
